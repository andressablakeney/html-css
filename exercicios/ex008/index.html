Principais formatacoes semanticas

Negrito / Destaque

Nesta frase temos um termo em negrito usando <b> (nao semantico)

Nesta frase tenho um termo em destaque usando <strong> 👍

Italico / Enfase

Nesta frase temos um termo em italico usando <i> (nao sematico)

Nesta frase temos um termo em italico usando <em> 👍

Mark

Podemos criar tambem um texto marcado com a tag <mark>

Podemos criar tambem um texto marcado e com um pouco de css podemos mudar a cor da tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

O <big> que deixava o texto maior (funciona ainda mas é obsoleto) esta depreciada pela HTML5, mas o <small> ainda existe

Texto deletado e inserido

Podemosmarcar um texto como deletado para fazer com que ele seja lido mas nao considerado

Podemos marcar texto como inserido para dar uma enfase e indicar que ele foi adicionado depois

Texto sobrescrito

Podemos inserir coisas do tipo x20+3

Texto subscrito

Podemos inserir coisas do tipo H2O
